# Placeholder Images

This directory contains placeholder images for the AI Dress Designer application.

## Usage

These images are used throughout the application for:
- Hero section background
- Sample gallery carousel
- Design previews

## Image Specifications

- **Hero Image**: 1200x1600px (3:4 aspect ratio)
- **Gallery Images**: 600x800px (3:4 aspect ratio)
- **Format**: JPEG or PNG
- **Optimization**: Images should be optimized for web

## Replacing Placeholders

To replace these placeholders with actual images:
1. Add your images to this directory
2. Update the image references in the components
3. Ensure images are properly optimized for web delivery
